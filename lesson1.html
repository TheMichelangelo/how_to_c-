1.Як встановити собі sqlite на linux?
How to install sqlite on linux?

SQLite — полегшена реляційна система керування базами даних. Втілена у вигляді бібліотеки, де реалізовано багато зі стандарту SQL-92. С ирцевий код SQLite поширюється як суспільне надбання (англ. public domain), т обто може використовуватися без обмежень та безоплатно з будь-якою метою. Фінансову підтримку розробників SQLite здійснює спеціально створений консорціум, до якого входять такі компанії, як Adobe, Oracle, Mozilla, Nokia, Bentley і Bloomberg.
Особливістю SQLite є те, що вона не використовує парадигму клієнт-сервер, тобто рушій SQLite не є окремим процесом, з яким взаємодіє застосунок, а надає бібліотеку, з якою програма компілюється і рушій стає складовою частиною програми. Таким чином, як протокол обміну використовуються виклики функцій (API) бібліотеки SQLite. Такий підхід зменшує накладні витрати, час відгуку і спрощує програму. SQLite зберігає всю базу даних (включаючи визначення, таблиці, індекси і дані) в єдиному стандартному файлі на тому комп'ютері, на якому виконується застосунок. Простота реалізації досягається за рахунок того, що перед початком виконання транзакції весь файл, що зберігає базу даних, блокується; ACID-функції д осягаються зокрема за рахунок створення файлу-журналу.(Детальніше https://uk.wikipedia.org/wiki/SQLite)
Для того щоб встановити собі sqlite, необхідно виконати декілька кроків:

  • Перейти на сторінку завантаження SQLite , і завантажити sqlite-autoconf-*.tar.gz з версії з кодом(source code section)
  • Перейти в папку з завантаженням і розпакувати архів виконавши в консолі команду tar xvfz sqlite-autoconf-<версія архіву>.tar.gz
    Наприклад
  • Перейти в папку з розпакованим архівом cd sqlite-autoconf-<версія архіву>
    Наприклад
  • Вказати місцем встановлення загальну папку з бібліотеками ./configure --prefix=/usr/local
  • Зібрати з вихідного коду програму встановлення виконавши команду make
  • Встановити собі базу данних командою sudo make install

2.Як користуватись sqlite?
How to use sqlite?

sqlite база данних без клієнта, вся робота з нею відбувається з консолі. Для того щоб почати роботу необхідно виконати команду sqlite3 в консолі



До базових команд належать:
  • .help - інфорація про команду/програму
  • .exit - вихід
  • .backup ?DB? FILE - зберегти базу данних в файл
  • .dump ?TABLE? - збеерегти дані таблиці в окремий файл
  • .import FILE TABLE - імпортувати данні з файлу в таблицю

Щоб створити нову базу данних/відкрити існуючих треба виконати команду sqlite3
Командою .databases можна побачити список всіх таблиць в бд
